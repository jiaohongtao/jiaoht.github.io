zookeeper连接不上/Session 0x0 for server null, unexpected error, closing socket connection

发表于 2019-12-26 更新于 2020-07-21 分类于 Linux
本文字数： 166 阅读时长 ≈ 1 分钟

好像是Linux重启导致zookeeper端口出了问题

设置默认接口区域，立即生效无需重启（这个好像是重置所有端口，慎用!!!，不过我确实是用这个解决了,然后重新打开需要的端口）
firewall-cmd --set-default-zone=public

打开端口命令：

firewall-cmd --add-port=端口号/tcp
# zookeeper # Linux端口
